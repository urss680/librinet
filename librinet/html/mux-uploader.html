01 · Genera la URL de subida desde tu terminal
⚠ Por qué no funciona llamar a la API directamente El endpoint de creación de Mux bloquea llamadas desde el navegador (CORS). Ejecuta el siguiente comando con tus credenciales para obtener la URL de subida:
Terminal — reemplaza TOKEN_ID y TOKEN_SECRET
curl -X POST https://api.mux.com/video/v1/uploads \
  -u "TOKEN_ID:TOKEN_SECRET" \
  -H "Content-Type: application/json" \
  -d '{"cors_origin":"*","new_asset_settings":{"playback_policy":["public"]}}'

La respuesta devuelve un JSON. Copia el valor de data.url y pégalo abajo.
Credenciales en dashboard.mux.com → Access Tokens · permiso necesario: Mux Video → write.
